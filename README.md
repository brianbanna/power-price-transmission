# The Price of Wind and Sun

A scrollytelling visualisation of how German solar output shapes electricity prices across Switzerland, France, Italy and Austria. Built for COM-480 Data Visualization at EPFL.

## Project

We work from 301,391 hourly observations of the European electricity market between January 2024 and June 2025. The narrative follows one sunny Sunday across five connected bidding zones (Switzerland, Germany, France, Italy, Austria). It then releases the reader into an explorer that covers the full eighteen months of hourly prices and generation mix.

The project has four parts:

- A seven-step narrative scroll built on a shared dark-themed map.
- An interactive explorer with a timeline scrubber, click-to-inspect sidebar, and play / pause controls.
- A Python preprocessing pipeline that turns the raw ENTSO-E CSV into hourly JSON aggregates.
- A 17 KB TopoJSON of the five focus countries, built from Natural Earth (via `world-atlas`) and committed to the repo.

## Dataset

[European Electricity Price and Generation 2024-2025](https://transparency.entsoe.eu/), from the ENTSO-E Transparency Platform. 301,391 hourly rows across 23 European bidding zones, with day-ahead prices and generation by source. The raw file lives at:

```text
data/entsoe_data_2024_2025.csv
```

It is never modified. Every derived artefact under `docs/data/processed/` is regenerated by the scripts in `scripts/`.

## Running the site locally

The site is static HTML + ES modules + D3 with no build step. Python 3.9+ is needed for the preprocessing pipeline.

```bash
# Python environment
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt

# Rebuild the five-country map (optional, already committed)
.venv/bin/python scripts/build_topojson.py

# Sanity-check the dataset against the reported figures
.venv/bin/python scripts/explore.py

# Serve the site from the repo root
python3 -m http.server 8000
# open http://localhost:8000/docs/
```

## Milestone 2 summary

The prototype runs end to end. Seven narrative steps cover the midnight baseline, the dawn solar ramp, the 13:00 peak where Switzerland clears at -€145.12 per MWh below Germany, a calendar heatmap of every hour in the eighteen-month window, Germany's merit-order generation stack on the showcase day, the monthly duck-curve evolution from 2024 to 2025, and a five-country small-multiples comparison. The explorer that follows has a timeline scrubber, a click-to-inspect country sidebar with a generation stack and daily profile, play / pause with 1x, 2x and 4x speed, and a colour toggle between price and renewable share.

The design document under [`milestone_2/`](milestone_2/) covers the project goal, the exploratory findings that shaped the narrative, screenshots of each step, the MVP and stretch-ideas split, and the course lectures that support each component.
