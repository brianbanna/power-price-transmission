## What's inside

A static website built from 301,391 hourly observations of the European electricity market between January 2024 and June 2025. The narrative follows one spring Sunday across the five tightly coupled bidding zones at the heart of central Europe — Switzerland, Germany, France, Italy and Austria — and then opens into a free-exploration dashboard where the user can scrub across 18 months of hourly prices and generation mix.

The project has four parts:

- **Narrative scroll** — a seven-step sequence with a dark-themed map of the five focus countries as the always-visible canvas.
- **Interactive explorer** — timeline slider, flow arrows inferred from price differentials between neighbouring markets, and side-by-side generation stacks.
- **Pre-processed data** — hourly JSON aggregates built from the raw ENTSO-E CSV via a reproducible Python pipeline.
- **Map geometry** — a 17 KB TopoJSON of the five focus countries, built once from Natural Earth (via `world-atlas`) and committed to the repo.

## Dataset

[European Electricity Price and Generation 2024–2025](https://transparency.entsoe.eu/), sourced from the ENTSO-E Transparency Platform. 301,391 hourly rows across 23 European bidding zones with day-ahead prices and generation by source. The raw file lives at:

```text
data/entsoe_data_2024_2025.csv
```

The raw CSV is treated as immutable — all derived artefacts are regenerated by the scripts under `scripts/`.

## Running the site locally

The site is static HTML + ES modules + D3 with no build step. You need Python 3.9+ for the data pipeline and any static file server for the page.

```bash
# 1. Python environment for the data pipeline
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt

# 2. (Optional) Regenerate the 5-country map topology
.venv/bin/python scripts/build_topojson.py

# 3. Sanity-check the raw dataset against the headline findings
.venv/bin/python scripts/explore.py

# 4. Serve the site (from the repo root, not from src/)
python3 -m http.server 8000
# then open http://localhost:8000/src/
```

## Milestone 2 — functional prototype

The site runs end-to-end. The narrative scroll covers all seven steps on a single sticky-map canvas: midnight baseline, dawn solar ramp, the 13:00 peak moment where Switzerland crosses below Germany at −€145.12/MWh, the year-in-one-view calendar heatmap, the generation-stack merit-order collapse, Germany's duck-curve evolution 2024→2025, and the five-country small-multiples comparison. The scroll then releases into an interactive explorer with a timeline scrubber, click-to-inspect country sidebar, play/pause with 1×/2×/4× speed control, and a price / renewable-share colour toggle.

The two-page design document under [`milestone_2/`](milestone_2/) describes the project goal, the exploratory-analysis findings that motivated the narrative, sketches for each scroll step, the MVP / stretch-ideas split, and the tools and course lectures that support each visualisation component.
